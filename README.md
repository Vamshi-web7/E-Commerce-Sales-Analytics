## Overview

This project analyzes e-commerce sales and customer transaction data using advanced SQL techniques and Power BI dashboards to generate business insights.

## Tools Used

* SQL
* Power BI
* Excel

## Advanced SQL Concepts Used

* Joins
* CTEs
* Window Functions
* Subqueries
* Views
* Stored Procedures
* Ranking Functions

## Key Analysis Performed

* Customer segmentation
* Revenue trend analysis
* Customer retention analysis
* Product performance analysis
* Regional sales analysis

## KPIs Tracked

* Total Revenue
* Monthly Sales Growth
* Customer Retention Rate
* Top Selling Products
* Revenue Per Customer
